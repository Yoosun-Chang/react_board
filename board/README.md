# react_board


>💡 **목표 : 나만의 React App 만들기**

- 요구사항에 맞는 기능을 포함하여 나만의 React App 을 만들어봅니다.
- 기획이 창의적일 필요는 없습니다. 필수 기능을 구현해보며 동작 원리를 이해하려 노력해보세요.
<br/>

>✔️ **과제를 진행하며 체크할 부분**
- 상태관리 ( 유지 / 초기화 ) 가 잘 되어있나요?
- 각 컴포넌트의 재사용성이 높나요?
- 예외처리가 미흡한 부분은 없나요?
<br/>
